Class GupshupPartnerTokenApiClient

Gupshup Partner Token API Client

Provides methods to interact with the Gupshup Partner Token API for managing app access tokens, partner apps, app creation, and authentication links.

Example

const client = new GupshupPartnerTokenApiClient({
  partnerToken: 'your-partner-token',
  debug: true
});

const accessToken = await client.getAccessTokenForApp('app-id');
